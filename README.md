# Student-s-Impact-Dashboard
## 📌 Project Overview

This project is a **Student Performance Dashboard** built using **Microsoft Excel** and **Power BI** to analyze academic data. It helps track student performance, attendance, and subject-wise scores through interactive and easy-to-understand visualizations.

---

## 🚀 Features

* 📊 Overall student performance analysis
* 📈 Subject-wise marks breakdown
* 📅 Attendance tracking
* 🏆 Identification of top and low performers
* 🔍 Interactive filters and drill-down insights

---

## 🛠️ Tools Used

* Microsoft Excel – Data preparation and cleaning
* Microsoft Power BI – Dashboard creation and visualization

---

## 📊 Key Metrics

* Total Students
* Average Marks
* Highest & Lowest Scores
* Attendance Percentage
* Subject-wise Performance

## ⚙️ How to Use

1. Open the Excel file and update student data
2. Open the `.pbix` file in Power BI
3. Click **Refresh**
4. Use filters to explore the dashboard
